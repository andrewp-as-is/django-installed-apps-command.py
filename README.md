### Installation
```bash
$ [sudo] pip install django-installed-apps-command
```

##### `settings.py`
```python
INSTALLED_APPS = [
    ...
    'django_installed_apps_command',
    ...
]
```

#### Examples
```bash
$ python manage.py installed_apps
```
